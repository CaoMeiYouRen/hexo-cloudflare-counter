> 兼容 LeanCloud Counter API 的 Hexo 阅读量统计后端服务，支持 Cloudflare Workers、Docker 和 Node.js 自托管，内置 D1/SQLite 存储方案。

[https://github.com/CaoMeiYouRen/hexo-cloudflare-counter#readme](https://github.com/CaoMeiYouRen/hexo-cloudflare-counter#readme)


## 📦 依赖要求


- node >=18

## 🚀 安装

```sh
npm install
```

npm run start
